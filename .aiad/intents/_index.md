> Commande : `/sdd-intent` dans Claude Code

## Statuts possibles

- **draft** — Intent en cours de rédaction
- **validé** — Intent validé par le PM, prêt pour spécification
- **en-cours** — SPECs en cours de rédaction/implémentation
- **livré** — Toutes les SPECs liées sont livrées
- **archivé** — Déplacé dans `archive/`
